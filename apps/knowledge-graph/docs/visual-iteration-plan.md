- 首页直接说明项目用途。

- 从旧的三栏控制台改成更清楚的页面结构。

## 下一步

- 点击节点后打开详情。
- 默认展示入门路径。
- 控制项不超过 3 个。

验收：

- 用户 5 秒内知道这里能点。
- `bun run typecheck` 和 `bun run build` 通过。

### 2. 做详情抽屉

- 展示一句机制解释。
- 展示对应源码路径。

验收：

- 点击任一节点能看到详情。
- 详情文字短、具体、无禁用词。
- `Esc` 能关闭详情。

### 3. 做路径和筛选

- 支持入门、上下文、安全、进阶四条路径。

验收：

- 路径切换后节点明显变化。
- 窄屏不遮挡内容。

### 4. 做视觉检查

- 截桌面图和窄屏图。
- 检查标题不重叠。
- 检查深色图块首屏可见。
